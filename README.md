# Python Portfolio

A collection of Python projects built to practise 
object-oriented programming — classes, inheritance, 
and subclasses.

## Projects

| Project | Description |
|---|---|
| RPG Battle Game | Turn-based combat game with class selection and enemy progression |
| Bank Account | Savings and checking accounts with interest and overdraft logic |
| Library System | Book inventory with checkout and return tracking |
| Online Store | Product catalogue with category subclasses and discount system |
| Virtual Pet | Pet simulation with hunger, energy and happiness stats |

## Concepts covered
Object-oriented programming · Inheritance · Method overriding · 
Input validation · Game loops
